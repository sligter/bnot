使用go+react开发一个层次化的笔记应用程序，专注于建立大型个人知识库。包含以下功能:
1.本地存储Markdown笔记
2.丰富的所见即所得笔记编辑功能，包括带有 Markdown 自动格式化功能的表格，图像和数学
3.支持编辑使用源代码的笔记，包括语法高亮显示
4.笔记之间快速导航，全文搜索和笔记聚焦
5.无缝笔记版本控制
6.支持嵌入多媒体文件，code, pdf，excel, ppt等视频，音频，图像等
7.公开地分享（发布）笔记到互联网
8.针对智能手机和平板电脑进行优化的用于移动设备的前端
9.夜间主题
10.Markdown 导入导出功能
11.使用网页剪藏轻松保存互联网上的内容
12.包含登录，修改密码等安全功能
给出技术选型，架构设计，项目结构，各个功能模块的完整代码
